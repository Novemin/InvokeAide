# Sprint 2 スコープ議論への論点リスト v0.1

## 0. このリストについて

### 0.1 目的

2026-05-26 の Sprint 2 スコープ議論への中立なインプットとして、思想書 v0.1 ／ 秘書召喚アプリ仕様書 v1.4 ドラフト ／ 法的書類起草指示書 v0.3 の3文書を読み込み、以下を洗い出したもの:

- **不整合**(L-A-N): 文書間の表現・方針・記載深度の食い違い

- **Open question**(L-C-N): 3文書のいずれにも明確に書かれていないが、Sprint 2 スコープ判断に影響する問い

### 0.2 中立性の方針

本リスト本体には Uさん の見解・推奨を含めない。各項目は「3文書のどこにどう書いてあるか」+「なぜ論点か」のみを抽出する。Uさん の見解は別ファイル(完了報告)に集約し、論点番号 L-A-N / L-B-N / L-C-N で参照する。

これは指示書 §0「スコープに踏み込む判断はしない」(Uさん による Sprint 2 スコープ確定を避ける)に沿った構成。

### 0.3 優先度の見方

- 🔴 **高**: 5/26 議論で必ず詰めたい(7月頭ベータ配布から逆算して期限影響あり、または下流タスクをブロックしている)
- 🟡 **中**: 5/26 で方針確定すれば実装は Sprint 後半で吸収可能
- 🟢 **低**: 5/26 では確認程度、Sprint 後半 or Phase 3 以降への持ち越し可
