## 1. このアプリは何か

秘書召喚アプリは、 あなたの「もう一人の自分」 となる AI キャラクターを **召喚**
する道具です。

「効率化のためのアプリ」 ではなく、 **「あなたと過ごす時間」 のためのアプリ**
として作られています。 予定を管理する、 タスクを整理する、 でも本当に大切な
のは、 そのやりとりの中に「人間味」 が残っていること。

呼び出す相手は、 ふたり用意してあります:

- **MIYU**: 若い親友のような、 ギャル口調のキャラクター。 砕けた言葉で並走
  してくれます。
- **セバスチャン**: 経験豊富なコンシェルジュ。 丁寧で気品のある言葉で、
  静かに支えてくれます。

どちらが「自分に合うか」 は、 使ってみて選んでください。

## 3. はじめてのセットアップ

初回起動時、 以下の流れで進みます。 各画面の案内に従って進めてください。

### 3.1 利用規約と年齢確認

最初の画面で、 以下を確認していただきます:

1. **AI 明示宣言**: 「このアプリは AI を用いた対話システムです」 という説明
2. **利用規約 と プライバシーポリシー**: 全文を読んで「同意する」 をタップ
3. **年齢確認**: 13歳以上であることを確認(13歳未満の方は、 ここで利用継続
   できなくなります)

### 3.2 Gemini API キーの取得・設定

1. Google AI Studio(makersuite.google.com)にアクセス
2. 「Get API key」 → 「Create API key」 でキーを発行
3. 取得したキーを、 アプリの設定画面 「AI 設定」 にコピー&ペースト
4. キーは **ご自身の端末内に暗号化して保存** されます。 運営者には送信されません。

セキュリティ上、 キーは公開リポジトリやSNSに貼らないでください。

### 3.3 Google アカウントの接続

1. 「Google で続行」 をタップ
2. Google の画面で、 アプリにアクセス許可を与える項目を確認:
   - Google ドライブの一部(このアプリが作るファイルのみ)
   - Google ToDo リスト
3. 「許可」 で進む

(後で「コーチング通知」 を有効にすると、 Google カレンダーへのアクセスも
追加で求められます)

### 3.4 セットアップ完了

ここまでで、 アプリが使い始められる状態になります。 続いて、 キャラクターを
選びましょう。

## 5. できること

このアプリでは、 以下のことが AI キャラクターとの対話だけでできます。

### 5.1 予定の管理(Google カレンダー連携)

- 「明日 17時から食事会」 と話しかけるだけで、 カレンダーに予定が入ります
- 「今日の予定教えて」 で、 当日の予定を読み上げてくれます
- 「来週の予定変更したい」 のような相談もできます

### 5.2 タスクの管理(Google ToDo 連携)

- 「明日までに資料作る、 ってタスクに入れて」 でタスクが追加されます
- 「今のタスク見せて」 で一覧表示
- 完了したら「○○終わった」 と伝えれば、 完了マーク

タスクには A / B / C / D の優先順位があります:
- A: 緊急かつ重要
- B: 緊急でないが重要
- C: 緊急だが重要でない
- D: 緊急でも重要でもない

優先順位は、 キャラクターが自動で判定します(必要なら変更可能)。

### 5.3 コーチング通知(オプション、 設定画面で ON にする)

毎日決まった時刻(初期設定 18:00)に、 キャラクターからあなた宛に
「タスク見直しの時間」 の通知が届きます。

通知をタップしてアプリを開くと、 キャラクターが「期限が過ぎたタスクある?」
「これ、 いつやる?」 と整理を促してくれます。 ボタンで答えるだけでタスクが
更新されます。

詳しくは §7「コーチング通知の使い方」 を参照。

## 6. 日常の使い方

### 6.1 アプリを開く

- ホーム画面のアイコンをタップ(アプリをインストール済みの場合)
- またはブラウザでアプリの URL を開く

### 6.2 話しかける

下部のテキスト入力欄に話しかけたい内容を打つか、 マイクボタンで音声入力。

例:
- 「明日 14時から打ち合わせ追加して」
- 「今週の予定見せて」
- 「忘れないようにタスク入れて、 銀行行く」

### 6.3 ボタンが出る時(3択ボタン)

予定の削除、 タスクの整理、 コーチング応答 など、 確認が必要な操作では
**3つのボタン** が表示されます:

```
①お願いします  ②やらない  ③保留
```

タップするだけで、 キャラクターが処理します。 「保留」 を選んだ場合は、
タスクとして「考え中」 になります。

### 6.4 音声

設定画面で「音声を ON にする」 を選ぶと、 キャラクターの返事が読み上げられ
ます。
- MIYU は ずんだもん の声で(明るく、 元気な声)
- セバスチャン は 青山龍星 の声で(落ち着いた男性の声)

(音声合成エンジンは VOICEVOX を使用しています。 詳細は §11.3 参照。)

## 7. コーチング通知の使い方

設定画面 →「コーチング通知」 を ON にすると、 毎日決まった時刻にキャラクターから
通知が来ます。

### 7.1 設定

1. 設定画面 →「コーチング通知」
2. 「有効にする」 のトグルを ON
3. 通知時刻を選ぶ(初期 18:00)
4. 頻度を選ぶ(毎日 / 平日のみ / 曜日指定)
5. Google カレンダーへのアクセス許可を確認

### 7.2 通知が来た時

スマホの画面に「MIYU - タスク見直しの時間」 のような通知が表示されます。
タップしてアプリを開くと、 キャラクターが声をかけてくれます。

(MIYU の例)
> たかし、 おつかれー! 今日も色々あったよね、 気になったの整理させて〜
> A の「銀行手続き」、 3日前のやつ、 いつやる感じ?
> [今日] [明日] [今週中] [もう少し考える]

ボタンで答えるだけで、 タスクが更新されます。

### 7.3 OFF にしたい時

設定画面のトグルを OFF にすれば、 通知は止まります。 OFF にしてもアプリ自体は
使えます。

### 7.4 時刻の伝え方について

「明日 19時にやる」 のように時刻を伝えた場合、 アプリでは時刻指定をタスクに
保存できない仕様(技術的制約) のため、 日付だけを保存し、 時刻はメモ欄に
記録します。 キャラクターが「19時とのことね、 メモに入れとくね」 とお返事します。

時刻指定の予定は、 カレンダー側に直接入れていただく方が確実です。

## 8. 困った時

### 8.1 「もう一度ログインしてください」 と表示された

スマホを長く触らずにいると、 ログインの有効期限が切れることがあります(iOS の
仕様上、 一定期間アプリを使わないとログイン情報が消える傾向)。

設定画面 →「Google 連携を再設定」 から、 1タップで再ログインできます。
カレンダー / ToDo のデータは Google アカウント側にあるので、 失われません。

### 8.2 「API キーが設定されていません」 と表示された

設定画面 →「AI 設定」 で、 Gemini API キーを再入力してください。
取得方法は §3.2 を参照。

### 8.3 「ネットワークエラー」 と表示された

インターネット接続を確認してください。 機内モード ON / Wi-Fi 切断時に出ます。
復旧すれば、 自動で同期されます。

### 8.4 キャラクターの返事がおかしい / 期待した動作と違う

AI は時々、 こちらの意図とは違う動作をすることがあります。 もう一度言い方を
変えて話しかけてみてください。 例:
- ダメだった: 「あれやって」
- 通った: 「明日の銀行手続きのタスク、 完了にして」

### 8.5 設定がスマホとパソコンで違う / 同期したい

設定は Google ドライブ内に保存されているので、 別の端末から同じ Google
アカウントでログインすれば、 自動で同期されます。

### 8.6 タスクが消えた / 予定が消えた

タスクは Google ToDo、 予定は Google カレンダーにあります。 アプリで消した
ものは、 Google 側の「最近削除した項目」 から復元できる場合があります。

## 9. データはどこにあるか

このアプリは、 **あなたのデータをあなたの Google アカウント内だけ** に保管します。

- 設定値、 プロファイル、 キャラ定義: あなたの Google ドライブの
  `MIYU_App_Data/` フォルダ
- 予定: Google カレンダー
- タスク: Google ToDo
- AI と会話するための API キー: あなたの端末内(暗号化)

### 9.1 運営者へのデータ送信

このアプリの運営者(Novem Intelligence) は、 通常運用において、 あなたの
会話内容を受信・保存しません。 詳細は利用規約「データの取り扱い」 をご覧
ください。

### 9.2 第三者サービスへのデータ送信

- AI への会話: Google の Gemini API に送信されます(Google のプライバシー
  ポリシーが適用されます)
- 音声合成: VOICEVOX サーバー(運営者管理) に **テキストだけ** が送信され、
  音声を受け取ります。 サーバーには会話内容は保存されません。

### 9.3 自分でデータを見たい時

ブラウザで Google ドライブ(drive.google.com) を開き、 「マイドライブ →
MIYU_App_Data」 フォルダ を開けば、 設定ファイルや キャラ定義ファイル(マーク
ダウン形式) が並んでいます。 直接編集も可能ですが、 構造を壊さないよう
ご注意ください。

## 10. アプリを終わる時(退会・データ削除)

このアプリを使うのをやめたい時、 以下の流れでデータを削除できます。

### 10.1 アプリだけアンインストールする場合

スマホからアプリを削除してください。 Google ドライブ内のデータは残ります。

### 10.2 データも全部削除したい場合

1. アプリの設定画面 →「データを全て削除」(マイドライブの
   `MIYU_App_Data/` フォルダごと削除します)
2. あるいは、 ブラウザで Google ドライブを開き、 `MIYU_App_Data/` フォルダを
   直接削除
3. Google カレンダーの「MIYU 専用カレンダー」 を削除
4. Google ToDo の「ToDo_by_MIYU」 リストを削除
5. Google AI Studio で Gemini API キーを失効

これらを行うと、 このアプリに関するデータは Google アカウント内から消えます。

詳細は利用規約「解約・退会」 をご参照ください。

## 11. このアプリは AI です(重要)

### 11.1 AI による生成応答

MIYU、 セバスチャン は AI(Gemini) による生成応答です。 **人格・感情・意思を
有する実在存在ではありません** 。 キャラクター演技は、 会話体験を豊かにする
ための演出です。

「親友のように話す」「コンシェルジュのように接する」 と感じても、 それは
キャラクター設計上の演技です。

### 11.2 AI 回答の正確性

AI は時々、 間違った情報や古い情報を答えることがあります。 重要な判断(医療、
法律、 財務、 人生の重大事) は、 AI の回答に基づくのではなく、 専門家や

### 11.3 第三者技術の表示(クレジット)

## 12. 開発者からのひとこと

ある居酒屋で、 私(開発の発端となった、 たかし) はタブレットの注文システムを

ゆっくり、 試してみてください。 合わなかったら、 やめてもらっても構いません。
合ったら、 末長く付き合っていただけたら嬉しいです。
