## 1. はじめに

本プライバシーポリシー（以下「本ポリシー」といいます）は、Novem Intelligence（以下「運営者」といいます）が提供する「秘書召喚アプリ／InvokeAide」（以下「本アプリ」といいます）における、ユーザーの情報の取り扱いについて定めるものです。

### 1.1 AI を利用したサービスである旨の明示

本アプリは、人工知能（AI）を用いた対話システムを含むサービスです。本アプリの応答や提案は AI による生成物であり、人間による直接の発話・判断ではありません。ユーザーは本アプリを利用する際、これを理解した上で利用するものとします。

### 1.2 本ポリシーの対象範囲

本ポリシーは、本アプリの利用に関連する個人情報・ユーザーデータの取り扱いを対象とします。本アプリと連携する第三者サービス（Google サービス、AI 提供事業者等）における情報の取り扱いについては、各事業者のプライバシーポリシーが適用されます。

---

## 2. 基本方針（BYOK モデルとデータ非保有構造）

### 2.1 運営者がユーザーデータを預からない設計

本アプリは、ユーザーの会話内容や個人データを、運営者側で保存・閲覧することを目的とした専用サーバーを運用していません。本アプリは、ユーザー自身の Google アカウント上のサービス（Google Drive、Google カレンダー、Google ToDo）およびユーザーの端末内で、データのやり取りが完結する設計を採用しています。

運営者は、通常運用において、ユーザーの会話内容、Google アカウントから取得したデータ、および端末内に保存された API キー等にアクセスする手段を持ちません。

### 2.2 BYOK（Bring Your Own Key）モデル

本アプリは、AI 機能の利用にあたり、ユーザー自身が AI 提供事業者（Google AI Studio 等）から取得した API キーを使用する BYOK モデルを採用しています。API キーは、ユーザーの端末内に暗号化して保存されます。運営者は API キーの内容を取得・保管しません。

---

## 3. 取得する情報

本アプリの利用にあたり、ユーザーの同意のもとで以下の情報にアクセスします。

### 3.1 Google アカウント情報（OAuth 認証）

| 項目 | 内容 |
|---|---|
| 認証情報 | Google アカウントの ID、メールアドレス、プロフィール情報（OAuth ログインに必要な範囲） |
| Google Drive | 本アプリが作成または選択したファイル（後述 §4 参照） |
| Google カレンダー | ユーザーの予定（後述 §4 参照） |
| Google ToDo | ユーザーのタスク（後述 §4 参照） |

### 3.2 ユーザーが本アプリに入力する情報

ユーザーが本アプリに対して入力する会話内容、設定情報、フィードバック等。

### 3.3 API キー（BYOK）

ユーザー自身が取得した AI 提供事業者の API キー。本アプリは端末内 IndexedDB に AES-GCM 方式で暗号化して保存します。

---

## 4. Google ユーザーデータへのアクセスと利用目的

本アプリは、OAuth 認証により、ユーザーから事前に同意を得た範囲でのみ Google ユーザーデータにアクセスします。各 OAuth スコープごとの利用目的は以下のとおりです。

### 4.1 各 OAuth スコープの利用目的（一覧）

| OAuth スコープ | アクセスするデータ | 利用目的 | 保存場所 | 第三者共有 |
|---|---|---|---|---|
| `openid` `email` `profile` | Google アカウント基本情報 | ユーザーの認証・本アプリへのログイン | ユーザー端末内 | しません |
| `https://www.googleapis.com/auth/calendar.events` | ユーザーが Google カレンダーに登録した予定（イベント）の閲覧・作成・更新・削除 | ユーザーが本アプリ経由で予定を確認・登録・変更するため | Google カレンダー（ユーザーのアカウント内） | しません |
| `https://www.googleapis.com/auth/tasks` | Google ToDo のタスクの閲覧・作成・更新・削除 | ユーザーが本アプリ経由でタスクを管理するため | Google ToDo（ユーザーのアカウント内） | しません |
| `https://www.googleapis.com/auth/drive.file` | 本アプリが作成またはユーザーが選択したファイルのみ（本アプリが管理するフォルダ `MIYU_App_Data/` 配下のファイル） | 本アプリの設定・会話履歴等の保存と読み出しのため | ユーザー自身の Google Drive | しません |

### 4.2 アクセス範囲の最小化

本アプリは、機能の実現に必要な最小権限のスコープを採用しています。例えば、Google Drive へのアクセスは、本アプリが作成または選択したファイルのみに限定する `drive.file` スコープを使用しており、ユーザーの Drive 内の他のファイルにはアクセスする手段を持ちません。

## 5. 第三者提供および第三者サービスの利用

### 5.1 AI 提供事業者への送信

本アプリの AI 機能を利用する際、ユーザーが入力した会話内容、および会話の文脈に必要な範囲で取得した Google ユーザーデータ（予定、タスク等）が、ユーザーが設定した AI 提供事業者の API（現時点では Google 社の Gemini API）に送信されます。これは、AI が適切な応答を生成するために必要な処理です。

AI 提供事業者に送信されたデータの取り扱いについては、当該事業者のプライバシーポリシーおよび利用規約が適用されます。ユーザーは本アプリを利用する前に、AI 提供事業者のプライバシーポリシーを確認するものとします。

参考リンク:
- Google Gemini API: https://ai.google.dev/terms

### 5.2 将来の AI 提供事業者変更の可能性

運営者は、技術的または商業的判断に基づき、本アプリが利用する AI 提供事業者を変更または追加する可能性があります（例: Anthropic Claude、OpenAI、ローカル LLM 等）。変更が行われる場合、運営者は合理的な事前通知を行い、ユーザーは利用する AI サービスを選択できる設計を維持するよう努めます。

### 5.3 Limited Use（限定利用）への準拠

本アプリが Google API から受け取った情報の利用は、[Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy)（Limited Use 要件を含む）に従います。本アプリは、Google API から取得したユーザーデータを、ユーザーが本アプリの機能を利用するという正当な目的の範囲を超えて利用しません。

### 5.4 運営者以外への第三者提供

運営者は、上記 5.1 に示す AI 提供事業者への送信を除き、ユーザーの情報を第三者に提供しません。法令に基づく開示要請がある場合を除きます。

## 6. データの保存場所と保存期間

### 6.1 保存場所

本アプリに関連するデータは、原則として以下の場所に保存されます。

| データ種別 | 保存場所 |
|---|---|
| 会話履歴 | ユーザー自身の Google Drive（`MIYU_App_Data/logs/conversations/` 配下） |
| アプリ設定・プロファイル | ユーザー自身の Google Drive（`MIYU_App_Data/` 配下） |
| API キー | ユーザー端末内（IndexedDB、AES-GCM 暗号化） |
| OAuth リフレッシュトークン | ユーザー端末内のみ（Google Drive には保存しません） |
| Google ユーザーデータ（予定・タスク等） | ユーザー自身の Google アカウント内 |

運営者は、これらのデータを保存・閲覧することを目的とした専用サーバーを運用していません。

### 6.2 保存期間

ユーザーのデータは、ユーザーが Google Drive または端末から削除するまで保存されます。運営者がデータの保存期間を一方的に決定することはありません。

### 6.3 将来追加される可能性のあるデータ収集について

運営者は、本アプリの品質向上・安定運用のため、将来的に以下のような匿名化された技術データの収集を導入する可能性があります。

- クラッシュレポート、エラー解析データ
- 個人を識別しない範囲の利用統計
- ライセンス認証・課金管理に必要な技術情報

これらを導入する場合、運営者は本ポリシーを改訂し、事前にユーザーに通知します。現時点では、これらの収集は行っていません。

## 7. ユーザーの権利

### 7.1 データの閲覧・編集・削除

ユーザーは、本アプリに関連する自身のデータを、いつでも以下の方法で閲覧・編集・削除できます。

- **会話履歴、設定情報**: ユーザー自身の Google Drive（`MIYU_App_Data/` 配下）にアクセスし、各ファイルを直接閲覧・削除できます。
- **Google カレンダー、Google ToDo のデータ**: ユーザー自身の Google アカウントから直接管理できます。
- **API キー**: 本アプリの設定画面から削除できます。

### 7.2 OAuth 連携の解除

ユーザーは、Google アカウントの設定画面（https://myaccount.google.com/permissions）から、本アプリの OAuth 連携をいつでも解除できます。解除後、本アプリは Google ユーザーデータにアクセスできなくなります。

### 7.3 アプリの削除

ユーザーは、本アプリを端末から削除することで、端末内に保存されたデータ（API キー、リフレッシュトークン等）を削除できます。Google Drive 内のデータは、ユーザー自身で削除する必要があります。

## 8. セキュリティ対策

運営者は、ユーザーの情報を保護するため、以下を含む合理的なセキュリティ対策を講じます。

- API キーの端末内 AES-GCM 暗号化保存
- 最小権限の OAuth スコープ採用（`drive.file` 等）
- 認証情報（リフレッシュトークン）の端末内限定保管
- HTTPS による通信の暗号化

なお、インターネット上の通信や電子的保存において、絶対的な安全性を保証することは困難です。運営者は合理的な対策を講じますが、ユーザー側でも API キーの厳格な管理、Google アカウントの二段階認証設定等、適切な情報管理を行うことを推奨します。
