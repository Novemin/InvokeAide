## 1. Sさん — 技術選定報告（Phase 1 Step 3）★週末の中心

### タスク

CLAUDE.md §5.2 で Sさん の判断に委ねられている技術選定を、根拠つきの報告書にまとめる。

選定対象（4項目）:

1. **フロントエンド**: React / Vue / Vanilla JS / その他
2. **バックエンド**: Node.js + Express 継続 / Apps Script 移行 / 別案
3. **LLM 抽象化レイヤー**: Gemini / Claude / OpenAI を切替可能にする設計（CLAUDE.md §4.1 — Gemini 固定にしない）
4. **音声認識 STT**: Whisper / Google Cloud Speech / Deepgram / その他

### 守る前提（CLAUDE.md §5.1 確定事項）

- 配信方式: PWA ＋ 必要に応じて Capacitor ラッパー

- 法的要件（CLAUDE.md §4.1）: データを送らない構造、年齢確認フロー、AI明示宣言、第三者AIサービスの抽象化 — 選定時に織り込む。

### 成果物

これは「Sprint 2 に入る前の宿題」。Sprint 2 で何を作るか（スコープ）の決定ではない。技術*選定*まで。
